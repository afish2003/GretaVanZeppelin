Musical Text Analysis

Is the up-and-coming band Greta Van Fleet actually that similar to the classic rockers of Led Zeppelin?

This project for Penn State Behrend's DIGIT 210 course aimed to use novel technologies in the XML stack to markup chord charts so that they could be analyzed by various Python and NLP programs. Our goal is to analyze their commonly used chord progressions and lyrical themes to discover connections, contrasts, and what makes each band unique!
